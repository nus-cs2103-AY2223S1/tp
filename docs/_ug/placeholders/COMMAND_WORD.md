The COMMAND_WORD is a text indicating a command word of a command. Refer to the [Command Format](#command-format) for more info.

```info
* COMMAND_WORD is **strictly** any of the following valid examples.
```

**Valid Examples:**

* exit
* help
* reset
* dec
* del
* edit
* find
* inc
* list
* new
* rmk
* sort
* view
* deletetag
* filtertag
* listtag
* newtag
* renametag
* tag
* untag

**Invalid Examples:**

* Anything else
