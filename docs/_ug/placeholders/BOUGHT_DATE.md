The BOUGHT_DATE is the date indicating when an item was bought.

```info
* BOUGHT_DATE must be in the following format (dd-mm-yyyy)
* BOUGHT_DATE can only have years between 1900 and 2300 inclusive
```

**Valid Examples:**

* 01-01-2022
* 28-09-2022
* 11-11-2022

**Invalid Examples:**

* 11-11-2400
* 11-11-1899
* 1-1-2022
* 01-1-2022
* 1-01-2022
* 01/09/2022
* 40-40-2022
* 1-9-22
* 1-nov-2202
